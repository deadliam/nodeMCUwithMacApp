TubZel is monitoring system of your bathroom.
Now you don't need to run and pull the door to find out whether it's open or not!

MAC OS APP:
	1. Execute Install.app

	2. Edit ip address in config file if needed
	3. Edit favorite song in config file


NODEMCU:
	1. Edit SSID and PASSWORD in main.py
	sta_if.connect("SSID", "PASSWORD")

	2. Upload main.py to nodemcu
